📚 **Study Dashboard**

A simple and modern Study Dashboard built using HTML to help track subjects, tasks, and focus time.

🚀 **Features**

- 📌 Subject tracking
- ✅ Task management
- ⏱️ Focus timer UI
- 🌙 Clean dark design

🛠️ **Tech Used**

- HTML
- CSS (inside HTML)
- JavaScript (inside HTML)


📂** Project File**

- "Dashboard.html"

💻 **How to Run**

1. Download or clone this repository
2. Open "Dashboard.html" in any browser
